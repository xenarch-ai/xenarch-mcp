# xenarch-mcp

MCP servers for the Xenarch payment network.

- **Agent MCP** — lets AI agents discover priced content and make micropayments
- **Publisher MCP** — lets publishers manage pricing and view earnings

## Install

```bash
# For AI agents
npx xenarch-agent-mcp

# For publishers
npx xenarch-publisher-mcp
```

## Development

```bash
npm install
npm run build
npm test
```
